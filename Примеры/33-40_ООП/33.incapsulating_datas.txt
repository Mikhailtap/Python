	класс - тип, описывающий набор свойств. Имеет структуру данных (функции или переменные). Членами класса могут быть функции (методы) и переменные (атрибуты).

Обращаться с помощью точечной записи: имя-класса.имя-метода() или имя-класса.имя-атрибута()

Class ClassName:
'''строка документации класса'''
объявление переменных класса
объявление методов класса

Объявление класса, в котором определяются его атрибуты и методы, является образцом, из которого могут быть произведены копии (экземпляры) класса. (имена классов с прописной буквы, а имена объектов - со строчной)
	Все переменные, объявленные внутри определения методов - локальны внутри класса, где были определены.
	Данный прием называется инкапсуляцией данных (данные доступны только локально, значит недоступны для остальной части программы) - первый принцип Объектно Ориентированного Программирования.

Ко всем свойствам класса можно обратиться локально, используя self. ПРИМЕР: self.sound, где sound - атрибут. Кроме того, все определения методов должны содержать self в качестве первого аргумента. То есть метод с именем talk выглядит как talk(self)
При создании экземпляра класса автоматически вызывается специальный метод __init__(self).
Если необходимо передать еще значения, то в скобки они могут быть добавлены последовательно.

Пример объявление класса:
class Critter:
'''Базовый класс для всех живых существ'''
count=0
def __init__(self, chat):
    self.sound=chat
    Critter.count+=1
def talk(self):
return self.sound

	count является переменной класса, доступна всем экземплярам данного класса. К ней можно обратиться с помощью записи Critter.count как внутри, так и извне класса
	метод __init__() - метод инициализации, который вызывается автоматически во время создания экземпляра класса. В данном случае инициализирует переменную экземпляра sound значением, переданным из аргумента chat, и увеличивает значение переменной класса count при каждом создании экземпляра этого класса.
	метод talk() объявлен как обычная функция, за исключением автоматически указываемого аргумента self (других значений не требуется). В данном случае метод возвращает значение, инкапсулированное в переменную экземпляра sound.
