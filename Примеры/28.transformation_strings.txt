До версии 3.0 строковые символы в языке Python хранились в соотв-ии с их числовыми кодами формата ASCII в диапазоне от 0 до 127, представляя только символы латинского алфавита без символов с ударением. Пример: а = 97
Благодаря Unicode все символы и алфавиты представлены числовым диапазоном от 0 до 1114111. Символы, которые находятся выше диапазона ASCII, могут требовать для хранения два байта, например шестнадцатеричное значение 0xC3 0xB6 для символа 6.
Для преобразования символов в кодировку Unicode существует метод encode() объекта str, а для обратного действия - метод decode().
В модуле unicodedata существует полезный метод name(), который отображает имя каждого символа в формате Unicode. Таким образом, нелатинские, а также символы с ударениями можно получать, используя их Unicode-имя, либо с помощью преобразования их шестнадцатеричного представления.
