Модуль pickle позволяет сохранить тип данных при переносе в файл
Процесс упаковки (консервации) данных сохраняет строковое представление объекта, которое впоследствии может быть распаковано к его первоначальному состоянию. Это наиболее распространенная процедура.
Метод dump() модуля pickle применяется для этого. С помощью dump() сохраняется в файл, а с помощью load() распаковывается из файла. Этот метод в качестве единственного аргумента применяет имя файла. Все хранится в двоичном виде.
Если в программе нужно будет проверить существование сохраняемого файла, то для этого в языке Python используется модуль os, предлагающий объект path, у которого есть метод isfile(), возвращающий True, если объект найден.
