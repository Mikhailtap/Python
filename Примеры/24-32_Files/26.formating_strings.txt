Функция dir() - использовать для получения имен всех функций и переменных, определенных в модуле. При этом имя нужного модуля указывать в скобках
!!!ВАЖНО!!! После вызова этой функции также появится атрибут __doc__.
Все, что начинается и заканчивается символами двойного подчеркивания - является зарезервированными объектами Python. С помощью dir() можно получить список всех имен и переменных, определенных по умолчанию в модуле __builtins__, к примеру print() и str.
Метод format() производит подстановки. Строка может содержать текстовые поля и поля для замены, куда будет вставляться текст из списка элементов, разделенных запятой. Каждое поле замены обозначается {}. Внутри может стоять порядковый номер заменяемого элемента.
Строки также можно форматировать с помощью %s (Как в С). Данный оператор будет помечать места в строке, куда будет вставляться текст из упорядоченного списка значений. !!!Для других типов данных при форматировании использовать %d - int, %c - char, %f - float
